# Gerenciador de Tarefas Moderno

Um aplicativo desktop simples para gerenciar suas tarefas do dia a dia. Construído inteiramente com Python e a interface gráfica nativa `tkinter`.

## 📌 Funcionalidades

* **Adicionar Tarefas:** Digite sua tarefa e clique em "Adicionar" ou pressione a tecla `Enter` do teclado.
* **Deletar Tarefas:** Selecione uma tarefa na lista e clique no botão vermelho "Deletar Tarefa Selecionada".
* **Lista com Rolagem:** Adicione quantas tarefas quiser; a lista possui uma barra de rolagem (scrollbar) integrada.
* **Validação Básica:** O aplicativo emite alertas caso você tente adicionar uma tarefa vazia ou deletar um item sem selecioná-lo antes.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **Tkinter:** Biblioteca gráfica padrão do Python.

## 🚀 Como Executar

Como o projeto utiliza apenas bibliotecas nativas do Python, a execução é imediata e não exige a instalação de pacotes externos.

1. Certifique-se de ter o Python instalado em seu computador.
2. Salve o código do projeto em um arquivo (por exemplo, `app.py`).
3. Abra o terminal ou prompt de comando na pasta onde o arquivo está salvo.
4. Execute o comando abaixo:

```bash
python app.py
